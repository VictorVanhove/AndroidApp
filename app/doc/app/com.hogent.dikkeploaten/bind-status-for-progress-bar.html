bindStatusForProgressBar

fun bindStatusForProgressBar(progressBar: ProgressBar, status: ApiStatus?): Unit

This binding adapter returns the ApiStatus of the network request for the ProgressBar. When the request is loading, the ProgressBar is set visible. When the request gets error or is finished, the ProgressBar is * set to invisible.

Parameters

progressBar - The ProgressBar when albums are loading.

status - The ApiStatus of the network request.
